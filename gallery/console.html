How to publish: click Connect repo folder and pick your local wellyzhang.github.io directory (grant write access). Add photos, set cities, then Save to repo — resized images land in the local img/gallery/ staging folder and gallery/photos.json is rewritten in place. Then push the images to R2 and commit the manifest: python3 gallery/upload.py && git add gallery/photos.json && git commit && git push. GitHub Pages rebuilds and the new pins appear on the globe, loading from R2. The image files stay out of the repo (img/gallery/ is gitignored).

Browser support: direct saving uses the File System Access API (Chrome / Edge / Opera). In other browsers this falls back to a downloadable ZIP you unzip at the repo root, then run python3 gallery/upload.py as above.

City lookup comes from a public cities database loaded on demand. Reverse-geocoding snaps GPS coordinates to the nearest city — exact enough for a city-level map. No GPS? Search for the city or type lat, lng directly.
